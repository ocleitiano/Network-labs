# 📚 Fundamentos de Redes de Computadores

Bem-vindo à seção **Fundamentos** do meu portfólio de Redes e Segurança.

Esta área reúne os principais conceitos teóricos que servem como base para os laboratórios práticos desenvolvidos ao longo deste repositório.

O objetivo é construir uma documentação organizada para consulta, revisão e demonstração da evolução dos meus estudos em Redes de Computadores e Segurança da Informação.

---

# 🎯 Objetivos

- Consolidar os principais conceitos de Redes.
- Servir como material de consulta rápida.
- Relacionar teoria com laboratórios práticos.
- Criar uma base sólida para estudos de Firewalls, NOC, SOC e Segurança da Informação.
